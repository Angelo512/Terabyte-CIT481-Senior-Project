Senior Project

As 2021 Spring semester started, our senior design group project also officially began.

This is Terabyte team member

At the end of last semester, we had an idea of creating a web calendar application that people use it to share their schedule so that they can make some fun events schedule together.

However, we were not really sure how we start it.
Our professor offered Alternate Project plan, it is for those who want to know how to start the project, like us. So, we decided to do the alternate.

Alternate project is split to three projects

  • Project 0
  • Project 1
  • Project 2

We need to assign a leader for each project, and each presentation date would be due of each project.
We have total four presentations on somewhere of:

  • Week of Feb 22 for 1st,
  • Week of Mar 22 for 2nd,
  • Week of Apr 19 for 3rd,
  • Final presentation would be either May 15th Saturday or May 20th Thursday

Our first goal is the first presentation in February.
We set regular group zoom meeting on every Saturday, after class CIT481 to prepare for it.
