Project Zero

May 12, 2021 · 1 min read

First, we established development environment.

  • Create Github repository for this project
    • Terabyte-CIT481-Senior-Project
  • Get AWS account and create IAM users
    • We use Lynn’s AWS account to create IAM users because he uses regular AWS account with educate credit.
  • Create Slack channel
    • Created “terabyte” channel including @Lisa and @Earl
  • Pick a leader for this project
    • Keiko became the leader

Project task

  • Purchase Domain
    • we purchased terabytecit.com from NameCheap.com
  • Set up AWS VPC (Virtual Private Cloud)
    • Create 3 public subnets and 3 private subnets
  • Set up AWS EC2 (Elastic Compute Cloud) as a Web server
    • Assigned EIP (Elastic IP address to the server)
  • Set up our domain to the Web server(EC2)
    • Use NameCheap.com DNS service to point to the EIP
  • Generate TLS Certificate to the site
    • User CertBot ACME client to attache the certificate to the domain
  • Create Ansible Playbook to setup/configure the server
    • Install apache and post custom index.html on the server
